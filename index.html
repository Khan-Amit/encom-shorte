▶️ Run encom-shorte (Real Code)

On Android (Termux):

pkg update && pkg install python git -y
git clone https://github.com/khan-amit/encom-shorte.git
cd encom-shorte
python prefilter.py

On Linux / Ubuntu:

sudo apt update && sudo apt install python3 git -y
git clone https://github.com/khan-amit/encom-shorte.git
cd encom-shorte
python3 prefilter.py

Continuous scanner (real‑time):

python -c "

⏹️ Stop the Scanner

Press Ctrl + C in the terminal where the script is running.

If that doesn't work, open a second terminal and run:

pkill -f python

Or close the terminal window entirely.
